Courses

Learn the essential skills needed to begin your journey into web development.

Introduction to HTML

This course introduces you to the fundamentals of HTML. You will learn how webpages are structured, how tags work, and how to create your first simple webpage.

Introduction to CSS

Learn how to style webpages using CSS. This course covers colours, fonts, layout, spacing, and how to make your webpage look polished and professional.

Basics of JavaScript

Understand how JavaScript brings interactivity to websites. This course teaches beginner‑level scripting, simple logic, and how browsers run JavaScript.

Building Your First Website

Combine your HTML, CSS, and JavaScript knowledge to create a complete webpage. This course guides you step‑by‑step through planning, designing, and building a simple website.
